1. 项目包含内容
    + 首页
        + 轮播图，tab切换
    + 列表页
        + 需要有分页
        + 至少五页
        + 每一页至少五条数据，最后一页无所谓
        + 如果你的列表页全是视频，只要要有两三个视频是可以真实播放
    + 详情页
        + 在列表页点击那个商品，看到那个商品详情
    + 购物车
        + 哪些商品加入到购物车里面了
    + 登陆注册
2. 请求数据
    + 可以写成 json 文件，直接本地请求
    + 可以使用 phpstudy，gulp 配置代理
    + 也可以不使用 gulp 直接使用 phpstudy
    + 写成本地的数组，进行渲染
